# eye-protection-assistant-
护眼助手（eye-protection-assistant）最初只是源于一个很简单的念头：好好保护眼睛这件事，不该总是被忽略。我们大都明白眼睛需要歇一歇，只是投入之后容易忘了时间。于是想做一个安静待在菜单栏里的小东西，不催促、不添乱，只是替你把"望向远处"这件小事，变成不用费心也能发生的事。eye-protection-assistant — a quiet macOS companion born from a simple wish: that resting your eyes shouldn't depend on remembering to. It sits quietly, so looking away becomes the easy default.
